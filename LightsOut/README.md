# LIGHTS OUT

## ルール

- マスの点灯を全て消しましょう
(F2に於いて点灯も消灯も等価なので全て点灯させてもよい)

- マスをプッシュすると光ったり消えたりする

- マスをプッシュする際に加わる変更は縦横であり、斜め以外である

- 変更の際、マスをはみ出たら無視

- なお変更とは、状態の反転であり、二進的なもの(点灯か消灯)であるとする

## サイトの仕様

このサイトのURLの末尾に「#n」(nは0より大きい非負整数とする)をつけるとn*nのマスのパズルが生成さる。
なお、生成されるパズルは、絶対的に、任意の状態で解が一意的に存在するもののみとする(存在しなければ自動的にn=3)。
今私が軽く調べたところによると、n=3,6,7,8,10,12,13,15,22,25,27のときは解が一意的に定まることが分かった(全く以て不思議である)。

具体的な計算を可視化したい

- 尚、具体的な計算とは、解の算出である(解の算出はもうできたので、後は高速化させたい)

- F_2の体で考えるものとする
