# LIGHTS OUT

## ルール

- マスの点灯を全て消しましょう
(F2に於いて点灯も消灯も等価なので全て点灯させてもよい)

- マスをプッシュすると光ったり消えたりする

- マスをプッシュする際に加わる変更は縦横であり、斜め以外である

- 変更の際、マスをはみ出たら無視

- なお変更とは、状態の反転であり、二進的なもの(点灯か消灯)であるとする

## 変更したい点

- 隣接行列などを、LaTeX記法で数式をHTML形式で表示できるMathJaxで、
具体的な計算を可視化したい

- 尚、具体的な計算とは、解の算出である(解の算出はもうできたので、後は高速化させたい)

- F_2の体で考えるものとする
